# ESP32 RFID Access Control System

A compact and robust access control system based on the **ESP32-WROOM-32E** microcontroller and the **RFID-RC522** reader. This project allows for controlling an electric strike, electromagnetic lock, or gate upon presenting an authorized card.

The system is designed to run on **12V DC**, making it easy to integrate with standard alarm and intercom systems.

---

## PCB Visualization

The PCB design (2-layer) prioritizes power stability and RF signal integrity.

### Key PCB Features:
* **RF Optimization:** Dedicated "Keep-out zone" under the ESP32 antenna to ensure maximum WiFi/Bluetooth range.
* **Stable Power Delivery:** Wide power rails for the 3.3V line to prevent voltage drops during high current peaks.
* **Safety:** Physical isolation slots routed near the relay to increase creepage distance and safety.

## Schematic & Functional Description

The system is divided into three main functional blocks.

### 1. Power Supply Section
The input stage accepts **12V DC**. It utilizes a modern **LMR51610** (SOT-23-6) step-down converter to efficiently lower the voltage to the **3.3V** required by the ESP32 and RFID module.
* **Input:** 12V DC (Screw Terminal J1).
* **Filtering:** Ceramic capacitors and a 1.5uH inductor ensure a stable output.

![Power Supply Schematic](<img width="1923" height="732" alt="obraz_2025-12-02_203330152" src="https://github.com/user-attachments/assets/2cf4b4b8-c2c5-42d2-8be3-19cc8297b05d" />)

### 2. MCU & User Interface
The core of the system is the **ESP32-WROOM-32E**. It handles the logic, wireless communication, and user feedback:
* **Feedback:** A Buzzer (BZ1) and 3 LEDs (Red, Yellow, Green) indicate access status.
* **Programming:** Exposed UART header (J3) with TX, RX, GND, 3.3V, IO0 (BOOT), and EN pins.

![MCU Schematic](<img width="1468" height="1161" alt="obraz_2025-12-02_203402684" src="https://github.com/user-attachments/assets/a3247a5e-af08-4306-b66a-adef96626155" />)

### 3. Actuator Output
The lock control mechanism is driven by a relay via an NPN transistor (Q1).
* **Output:** Screw Terminal J4 (NO - Normally Open contact).
* **Protection:** A flyback diode (D5) protects the transistor from voltage spikes generated by the relay coil.

##  Pinout & Connectors

| Designator | Type | Description |
| :--- | :--- | :--- |
| **J1** | Screw Terminal | **12V DC** Power Input |
| **J2** | Pin Header 1x8 | **RFID-RC522** Module Connector (SPI) |
| **J3** | Pin Header 1x5 | UART Programming Port (TX, RX, RST, BOOT, GND) |
| **J4** | Screw Terminal | Relay Output (Connect to Electric Strike) |

---

##  Bill of Materials (Key Components)

* **MCU:** ESP32-WROOM-32E (WiFi/BT Module)
* **Regulator:** LMR51610XF (DC-DC Step-Down)
* **Relay:** 12V Coil (Miniature SPST/SPDT)
* **Transistor:** NPN (e.g., BC817, SOT-23)
* **Inductor:** 1.5uH (Power inductor for the buck converter)
* **RFID:** RC522 Reader Module (connects to J2)
